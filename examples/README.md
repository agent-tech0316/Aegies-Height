# Aegies Height Examples

This folder is for height-project examples and probes. Agentech wrapper demos
have moved to `../../agentech_sdk/examples/`, and official FF SDK examples live
under `../../agentech_sdk/agentech_sdk/examples/`.

## Vision And Height

| Example | Purpose |
| --- | --- |
| `vision/height_calculator.py` | Distance and height math helpers. |
| `vision/human_height_live.py` | Live person/depth/status pipeline. |
| `vision/grid_laser_calibration.py` | Grid and laser calibration workflow. |
| `vision/tilt_telemetry_probe.py` | Robot pitch telemetry and image probe. |
| `vision/auto_aim_height.py` | Auto-aim loop for distance, centering, and pitch. |

Install `ff_sdk` from `../../agentech_sdk/agentech_sdk/wheels` before running
robot-control examples.
